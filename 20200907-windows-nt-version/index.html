Windows NT对应版本

本文最后更新于：September 7, 2020 pm

本文主要记录各个Windows NT版本对应的Windows版本。原维基百科链接可以点击这里查看。

Windows NT，新技术视窗操作系统（Windows New Technology）的简称，是美国微软公司1993年推出的纯32位操作系统核心。其基于OS/2 NT的基础构造。OS/2是由微软和IBM联合研制，分为微软的Microsoft OS/2 NT与IBM的IBM OS/2。由于双方在协作后来不欢而散，IBM继续向市场提供先前的OS/2版本；而微软则把OS/2 NT改名为Windows NT，并在1988年11月开始了对于“WinNT”（即第一代的Windows NT 3.1）的产品研发。在研发初期，Windows NT曾一度被认为将会是原先OS/2的3.0版本，但面世之后的Windows NT是一种纯32位操作系统，采用NT核心技术。后期更新的Windows NT支持32与64位两种版本。
