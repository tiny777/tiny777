在CentOS中安装VisualStudioCode

本文最后更新于：October 22, 2019 pm

在CentOS上使用yum repo安装VSC，实测CentOS7和CentOS8均可正常安装。

VSC的安装非常简单，因为微软官方有提供yum repo，因此我们直接添加repo然后使用yum安装即可，注意在CentOS8中的yum实际上已经被替换成了dnf，所以使用yum或者是dnf都是一样的。

VSC官网提供了非常详细的安装教程，需要的同学可以点击这里查看。下面只贴出安装记录。
